# WorkBuddy-Vocab 雅思智能词汇学习网站
轻量化单文件前端网页工具，无需后端、无需框架依赖，浏览器直接打开使用。

## 核心功能
1. 单词全自动解析
输入任意雅思词汇，一键自动输出全套备考资料：
- 中英分层释义（基础释义+雅思考场专用释义）
- 英美双音标 + 真人音频朗读播放
- 雅思真题同源考点例句
- 音节拆分读音记忆法
- 词根词缀字母记忆法

2. 专属生词收藏模块
基于浏览器 LocalStorage 实现本地持久化存储，收藏单词关闭页面不丢失，支持随时调取复习清单。

3. 跨端轻量化适配
单HTML文件架构，无复杂依赖，电脑、手机浏览器均可流畅访问，支持Github Pages一键部署线上演示。

## 使用方法
1. 下载仓库内 `index.html` 文件
2. 双击文件，用任意浏览器直接打开，即可使用全部词汇查询、记忆、收藏功能

## 项目工程化说明
代码开源托管于GitHub，规范版本提交记录，支持静态网页部署，可生成公开线上访问链接。
